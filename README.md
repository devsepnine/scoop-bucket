# Scoop Bucket for hibi-ai

Scoop bucket for [hibi-ai](https://github.com/devsepnine/hibi_ai) - TUI installer for Claude Code and Codex CLI configurations.

## Installation

Add this bucket:

```powershell
scoop bucket add hibi https://github.com/devsepnine/scoop-bucket
```

Install hibi-ai:

```powershell
scoop install hibi-ai
```

## Usage

After installation, run:

```powershell
hibi
```
